# WhatsApp Digital Catalog for Small Businesses

Responsive digital catalog for small businesses with WhatsApp ordering integration.

Designed for restaurants, clothing stores, beauty shops, coffee shops and local businesses that want a fast and simple way to showcase products and receive orders directly through WhatsApp.

## Features

- Responsive design  
- WhatsApp ordering integration  
- JSON-based products system  
- Reusable and modular structure  
- Easy branding customization  
- Dynamic product rendering  
- Netlify ready  
- Lightweight and fast

## Tech Stack

- HTML5
- CSS3
- JavaScript
- JSON
- Netlify
